<h2>Vue-demo(本项目只是demo，提供观看)</h2>

<p>目前用到了一些简单的功能：</p>

<p>1. Node.js安装(不多BB)</p>
<a href=" https://nodejs.org/en/download/"> https://nodejs.org/en/download/</a>

<p>2. 安装cnpm</p>

<pre><code>
	原因一个字儿：就是快.
	npm ||cnpm 需要装3.0||以上的版本
	npm install cnpm -g
</code></pre>

<p>3、安装vue-cli</p>
<pre>npm install -g vue-cli</pre>

<p>4、使用vue-cli初始化项目</p>
<pre>
	vue init webpack my-project
</pre>

<p>4.1 不会进目录，请出门左拐</p>

<p>5.进入目录 安装依赖</p>
<pre>
	刚说的cnpm 在这里会省时间
	cnpm install
</pre>
<p>6. 启动项目</p>
<pre>
	npm run dev
 </pre>

<p>7. 做掉项目</p>
<pre>
	rm -rf ../my-project
 </pre>
